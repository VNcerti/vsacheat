Hướng dẫn 4 bước

  • 1
    Tải Profile

    Cài đặt Profile
    Vào Cài đặt → Cài đặt chung → VPN & Quản lý thiết bị → Chọn profile vừa tải
  • 3
    Xác nhận cài đặt
    Nhấn "Cài đặt" → Nhập mật khẩu → "Cài đặt" lần nữa → "Xong"

    Xem UDID
    Quay lại trang web và tải lại trang để xem UDID
